# mirlim.blog

AI/AX 시대의 생각, 전문지식, 결과물을 공유하는 개인 블로그.
사람과 AI 에이전트 모두를 위해 설계.

## Tech Stack
- Astro (static + server hybrid)
- MCP Server (@modelcontextprotocol/sdk)
- Cloudflare Pages + Workers
- TypeScript
